Issue 7: Stanford course, tf.Transform, and Raspberry Pi: February 27, 2017

News

Preprocessing for Machine Learning with tf.Transform

Today we are announcing tf.Transform, a library for TensorFlow that allows users to define preprocessing pipelines and run these using large scale data processing frameworks, while also exporting the pipeline in a way that can be run as part of a TensorFlow graph.

Prophet: forecasting at scale

Facebook is open sourcing Prophet, a forecasting tool available in Python and R. Forecasting is a data science task that is central to many activities within an organization. For instance, large organizations like Facebook must engage in capacity planning to efficiently allocate scarce resources and goal setting in order to measure performance relative to a baseline.

Articles

How Zendesk Serves TensorFlow Models in Production

Zendesk is developing a series of machine learning products, the most recent of which is Automatic Answers. It uses machine learning to interpret user questions and responds with relevant knowledge base articles. This article describes the engineering architecture and lessons learned from running TensorFlow in production.

How we made TensorFlow run on a Raspberry Pi using Rust

A set of open-sourced scripts for cross-compiling TensorFlow onto the Rasperrby Pi, with support for calling TensorFlow code from Rust.

Tutorials

CS 20SI: Tensorflow for Deep Learning Research

This course will cover the fundamentals and contemporary usage of the Tensorflow library for deep learning research. We aim to help students understand the graphical computational model of Tensorflow, explore the functions it has to offer, and learn how to build and structure models best suited for a deep learning project. Course materials and slides are posted online, and there is a guest mailing list for questions.

7 simple steps to install TensorFlow on Windows

Step by step guide with screenshots and videos of installing a standard TensorFlow setup on Windows.

Getting started with TensorFlow

In this post we train a Tensorflow model to learn a certain linear function. That is, we provide the model with correct input/output examples until it figures out how to compute correct outputs for given inputs.

Projects

Image-to-Image Demo: Interactive Image Translation with pix2pix-tensorflow

A live web-based demo of pix2pix, a model for image to image translation. The demo is fast and clean, and the source is freely available on GitHub.

Fast PixelCNN++: speedy image generation

We speed up the image generation algorithm of PixelCNN++ by avoiding redundant computation through caching. Naive generation discards computation that can be re-used and performs additional computation that will not be used to generate a particular pixel.
